# Spell Correct
This is a __Spelling Correction__ app writtern using python.

## Algorithm
It is based on the edit distance method.

### How to Use
1. Simply clone files to a folder and enter following commands.
```
$ cd project-folder
python app.py
```
This will start the flask server at localhost 127.0.0.1 at port 8080

2. Go to browser and enter http://127.0.0.1:8080/spellcorrect/<Word you want to correct>
  
  This will return the spelling corrected word.
